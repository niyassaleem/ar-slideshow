# AR Slideshow

This project is an Augmented Reality (AR) Slideshow built entirely using HTML. It provides a visually engaging way to present slides with augmented reality features.

## Features

- 100% HTML-based design.
- Lightweight and easy to use.
- Supports modern browsers.

## Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/niyassaleem/ar-slideshow.git
   ```
2. Navigate to the project directory:
   ```bash
   cd ar-slideshow
   ```

## Usage

1. Open the `index.html` file in your browser.
2. Follow the instructions on the screen to interact with the AR Slideshow.
